A Web Application for Verification of Pathogen Eradication (AVPE) in Animal Colonies

Overview

This web application was designed as an aid for determining when an infected animal colony can return to normal operations. After the presence of an infectious agent has been identified in a colony, extensive efforts are made to eradicate the pathogen. Although the gold standard approach involves depopulation and re-population, this not commonly practiced. Typically a form of test and cull strategy is implemented. Testing is commonly performed through several waves of sampling. This AVPE is designed to assist colony managers in deciding how many consecutive negative tests are sufficient to lift the isolation status. Although an infectious agent event may follow a much more complex model, the model that we propose is likely to be conservative in that it provides confidence intervals for infection that are broader than those associated with more complex (and realistic) models.1 This AVPE is based on the following model assumptions.

Assumptions and Definitions

After the infection is thought to be eradicated, k waves of tests are performed with n tests performed in each wave. Each of these n x k test results is negative for capturing the presence of the infectious agent. The tests are typically performed either on individual animals or animal cages (hereafter referred to as cages). We assume that the probability of a cage testing positive is constant in each testing wave and that the test results are independent across cages. The likelihood of a cage testing positive may grow exponentially between testing waves. The time between consecutive test waves is constant.

Notation

k

is the number of test waves.

n

is the number of cages tested in each wave.

πi

is the probability that an individual cage will test positive on the i th test wave. (For example, π1 is the probability of a cage testing positive on the first wave.)

t

is the time between consecutive test waves.

r

is the growth rate of the pathogen over time, defined by the equation πi+1 = πi(1+r)t For example, if t = 1 (say one week between waves) and r = 0.2, then the probability that a cage will test positive will increase by 20% between consecutive test waves. Note that if r = 0, then the probability that a cage will test positive will be the same on all waves.

Input

k, n, r, and t.

Output

95% and 99% confidence intervals for π1, the probability that a cage will test positive on the first test wave.
